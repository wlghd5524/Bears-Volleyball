# SFML--Bears-Volleyball
Bears Volleyball

SFML 라이브러리(C++)를 사용한 오프라인 배구 게임

어렸을 적 학교 컴퓨터실에서 친구들과 재밌게 했던 '피카츄 배구' 게임을 모방하여 만든 게임

공이 몸에 부딪힐 수록 공이 빨라지게 설정하였고, 다이빙 기능도 구현하였다.
